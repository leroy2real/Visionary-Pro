Setup Guide

Get a free YouTube Data API v3 key in 5 minutes. Required for live data tools.

API Key
✓ API key saved successfully!
5-Step Walkthrough
1

Go to Google Cloud Console

Visit console.cloud.google.com and sign in with your Google account.

2

Create a New Project

Click "Select a project" → "New Project". Name it anything (e.g. "YouTube Tools") and click Create.

3

Enable YouTube Data API v3

Go to "APIs & Services" → "Library". Search for "YouTube Data API v3" and click Enable.

4

Create Credentials

Go to "APIs & Services" → "Credentials" → "Create Credentials" → "API Key". Your key will appear — copy it.

5

Paste Your Key Above

Paste the key in the field above and click Save. The green dot in the sidebar confirms it's active. You get 10,000 free units/day.
